DESCRIPCIÓN DEL PROYECTO
================================================================================

SIGID es una solución integral diseñada para optimizar la logística de activos
tecnológicos. El sistema facilita los procesos de picking y packing, asegurando
un control preciso del inventario y una gestión eficiente de los despachos en
entornos empresariales.

TECNOLOGÍAS UTILIZADAS:
    - Frontend: React.js, CSS3 (Glassmorphism), React Router
    - Backend: Node.js, Express.js
    - Base de Datos: MongoDB Atlas
    - Mobile: Android (Kotlin), Material Design 3
    - Control de Versiones: Git
    - Gestión de Proyecto: Scrum, Jira

FUNCIONALIDADES PRINCIPALES:
    ✓ Gestión de Activos Tecnológicos
    ✓ Control de Inventario en Tiempo Real
    ✓ Generación Automática de Guías de Despacho (PDF)
    ✓ Sistema de Autenticación Corporativa (MFA)
    ✓ Panel Administrativo con Métricas
    ✓ Panel Logístico para Operadores
    ✓ Aplicación Móvil Android Nativa

ESTADÍSTICAS DEL CÓDIGO:
    - Total líneas de código funcional: +1,400
    - Arquitectura: Cliente-Servidor (Decoupled Architecture)
    - Versión actual: 2.4.0-STABLE
    - Build: 2026.04.29
